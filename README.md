# 🔥 DIABOLIC PENINSULAR v1.0

[![Version](https://img.shields.io/badge/version-1.0-red)](https://github.com/Condor2026/Diabolic_Peninsular_V17)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.8+-green)](https://python.org)

Herramienta OSINT de código abierto que monitoriza **62+ periódicos digitales de la España peninsular** para detectar, clasificar y visualizar patrones delictivos (robos, estafas, narcotráfico, violencia...) por comunidades autónomas.

## 🚀 Características principales

- ✅ Scraping inteligente con detección automática de URLs rotas.
- ✅ Rotación de User-Agent para evitar bloqueos.
- ✅ Clasificación automática de delitos (12+ tipos).
- ✅ Detección de conexiones entre incidentes (posibles bandas).
- ✅ Visualización web interactiva con gráficos dinámicos.
- ✅ Exportación de datos a JSON y CSV.
- ✅ Menú terminal con 10 comandos avanzados.
- ✅ 100% legal y ético: solo datos públicos, sin almacenamiento de información personal.
